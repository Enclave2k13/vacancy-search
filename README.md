# Проект vacancy-search

## Описание:

Проект vacancy-search - это консольное приложение на Python для сбора и анализа вакансий с сайта hh.ru.

## Установка и запуск

1. Установите PostgreSQL
2. Склонируйте репозиторий
3. Создайте файл `.env` со своими данными (см. `.env.example`)
4. Установите зависимости: `poetry install`
5. `pip install curl_cffi`
6. Запустите программу: `poetry run python src/main.py`

База данных и таблицы создадутся автоматически.

## Тестирование:
```
pytest tests/ -v  # обычный запуск
pytest --cov=src --cov-report=html tests/  # с отчетом о покрытии
```

## Использование:

1. Запустите приложение в вашей консоли.
2. Вызовите необходимые функции.
